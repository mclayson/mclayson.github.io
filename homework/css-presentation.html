CSS Structure

@ rules

CSS is made of two different kinds of statements. The first are these @-rules. The @-rules are used for things like loading in more stylesheets or for adjusting styles based on specific requirements like screen width.

Rule Set

The bulk of CSS is made up of rule sets. These rule sets are made up of a selector and a declaration block which contains delcarations, each having a property and one or more values.

In the above example, the "p" is the selector. The selector determines what HTML elements will recieve the style. In this case all paragraph elements will revcieve the style changes.

Everything inside the curly brackets {} is the declaration block. The declaration contains the style declarations to be applied to the selector.

The individual lines within the declaration block are the style declarations. Each style declaration has one property and at least one value. The first declaration in this example uses the property "color", which changes the text color, and the value "red". So all paragraph elements in this example would have their text colored red.
